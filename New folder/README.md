```bash

```

### Development

```bash

```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Components Used

This project uses components inspired by ReactBits.dev:

- **SplitText**: Animated text component that reveals words one by one
- **AnimatedBackground**: Particle system for dynamic background
- **Confetti**: Falling confetti animation
- **FloatingElements**: Sparkle effects floating across the screen

## Customization

You can easily customize:

- Colors in `src/App.css` (gradient backgrounds)
- Text content in `src/App.jsx`
- Animation timings and delays
- Number of particles and confetti pieces

## Technologies

- React 18
- Vite
- Framer Motion
- CSS3 (Glassmorphism effects)
